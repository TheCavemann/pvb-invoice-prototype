# Manual testing reference

The mock data in `src/data/wallets.js` and `src/data/transactions.js` is generated by a
seeded PRNG (`mulberry32`), so it is identical on every reload/build. The sample values
below were pulled directly from that generated dataset and are guaranteed to exist —
use them to exercise the Wallets search page manually. The Wallets page "Search with"
dropdown supports six search types: Wallet ID, Account Number, Business Name, Branch
Name, Transaction Reference, and Session ID.

This file is for manual testing reference only; it is not part of the app.
